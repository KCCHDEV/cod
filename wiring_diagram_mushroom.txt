ระบบรดน้ำเห็ดนางฟ้าอัตโนมัติ - MH Sensor
==========================================

ESP32 Pin Configuration:
------------------------
GPIO 5  -> รีเลย์ปั๊มน้ำ (RELAY_PIN)
GPIO 36 -> MH Sensor AO (Analog Output)
GPIO 35 -> MH Sensor DO (Digital Output) 
GPIO 2  -> LED สถานะ (STATUS_LED)
GPIO 0  -> ปุ่มรีเซ็ต WiFi (WIFI_RESET_BUTTON)
GPIO 21 -> SDA (สำหรับ RTC - ถ้ามี)
GPIO 22 -> SCL (สำหรับ RTC - ถ้ามี)

Hardware Connections:
--------------------

1. MH Sensor (Soil Moisture Sensor):
   VCC -> 3.3V หรือ 5V
   GND -> GND
   AO  -> GPIO 36 (ESP32)
   DO  -> GPIO 35 (ESP32)

2. Relay Module:
   VCC -> 5V (หรือตามที่รีเลย์รองรับ)
   GND -> GND
   IN  -> GPIO 5 (ESP32)
   COM -> แหล่งจ่ายไฟปั๊มน้ำ (12V+)
   NO  -> ปั๊มน้ำ (12V+)

3. Water Pump:
   + -> รีเลย์ NO (Normally Open)
   - -> แหล่งจ่ายไฟ GND (12V-)

4. Status LED:
   + -> GPIO 2 (ESP32)
   - -> GND (ผ่านตัวต้านทาน 220Ω)

5. WiFi Reset Button:
   ขาหนึ่ง -> GPIO 0 (ESP32)
   อีกขา -> GND

6. RTC DS3231 (ถ้ามี - ไม่บังคับ):
   VCC -> 3.3V
   GND -> GND
   SDA -> GPIO 21 (ESP32)
   SCL -> GPIO 22 (ESP32)

Power Supply:
-------------
- ESP32: 5V หรือ 3.3V
- Relay: 5V (ตามที่รีเลย์รองรับ)
- Water Pump: 12V DC (หรือตามที่ปั๊มรองรับ)
- MH Sensor: 3.3V หรือ 5V

Mushroom System Logic:
---------------------
- รดน้ำเมื่อความชื้น < 60% (สูงกว่าผักชีฟลั่ง)
- หยุดรดน้ำเมื่อความชื้นถึง 85% (สูงกว่าผักชีฟลั่ง)
- ป้องกันการรดน้ำบ่อยเกินไป (ห่างกัน 20 นาที)
- จำกัดการรดน้ำวันละ 8 ครั้ง (มากกว่าผักชีฟลั่ง)
- ตารางรดน้ำ: 6:00, 12:00, 18:00 (3 ครั้ง/วัน)

Available Pins (สำหรับขยายระบบ):
-------------------------------
GPIO 4, GPIO 12, GPIO 13, GPIO 14, GPIO 15, GPIO 16, GPIO 17, GPIO 18, GPIO 19, GPIO 23, GPIO 25, GPIO 26, GPIO 27, GPIO 32, GPIO 33

Mushroom Growing Considerations:
-------------------------------
1. เห็ดนางฟ้าต้องการความชื้นสูง (60-85%)
2. รดน้ำบ่อยกว่าผักใบ (3 ครั้ง/วัน)
3. ระยะ Spawning: รดน้ำน้อย (30% น้อยกว่าปกติ)
4. ระยะ Fruiting: รดน้ำมาก (30% มากกว่าปกติ)
5. เวลากลางวัน: รดน้ำ 20% นานกว่า (เห็ดต้องการน้ำมากขึ้นเมื่อร้อน)

Status Levels (สำหรับเห็ดนางฟ้า):
-------------------------------
- เหมาะสม: ≥ 85% (สีเขียว)
- ดี: 70-84% (สีเขียวอ่อน)
- แห้งเล็กน้อย: 60-69% (สีส้ม)
- แห้งมาก: < 60% (สีแดง - ต้องรดน้ำทันที)

Notes:
------
1. เห็ดต้องการความชื้นสูงกว่าผัก เพื่อการเจริญเติบโตที่ดี
2. ระบบจะรดน้ำอัตโนมัติเมื่อความชื้นต่ำกว่า 60%
3. หยุดรดน้ำเมื่อความชื้นถึง 85% เพื่อป้องกันเชื้อรา
4. ระบบมีตารางเวลารดน้ำ 3 ครั้ง/วัน (เช้า กลางวัน เย็น)
5. หากไม่มี RTC ระบบจะใช้ uptime แทนเวลาจริง
6. ระบบมี Growth Stage Detection ปรับปริมาณน้ำตามระยะการเจริญเติบโต
