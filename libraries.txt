ESP32 RTC Relay Controller - Required Libraries
==============================================

Install these libraries in Arduino IDE:

1. RTClib by Adafruit
   - Author: Adafruit
   - Version: 2.1.1 or later
   - Description: Library for DS3231 RTC module
   - Install: Tools > Manage Libraries > Search "RTClib"

2. ArduinoJson by Benoit Blanchon
   - Author: Benoit Blanchon
   - Version: 6.21.3 or later
   - Description: JSON library for Arduino
   - Install: Tools > Manage Libraries > Search "ArduinoJson"

Built-in Libraries (no installation needed):
- WiFi (ESP32)
- WebServer (ESP32)
- Wire (I2C communication)
- Arduino.h (core Arduino functions)

Installation Steps:
==================

1. Open Arduino IDE
2. Go to Tools > Manage Libraries
3. Search for each library name
4. Click "Install" for each library
5. Restart Arduino IDE if needed

Alternative Installation (Manual):
=================================

If library manager doesn't work:

1. Download libraries from GitHub:
   - RTClib: https://github.com/adafruit/RTClib
   - ArduinoJson: https://github.com/bblanchon/ArduinoJson

2. Extract to Arduino/libraries folder:
   - Windows: Documents\Arduino\libraries\
   - Mac: ~/Documents/Arduino/libraries/
   - Linux: ~/Arduino/libraries/

3. Restart Arduino IDE

Board Configuration:
===================

1. Install ESP32 board support:
   - File > Preferences
   - Add URL: https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
   - Tools > Board > Boards Manager
   - Search "ESP32" and install

2. Select Board:
   - Tools > Board > ESP32 Arduino > ESP32 Dev Module

3. Configure Settings:
   - Upload Speed: 115200
   - CPU Frequency: 240MHz
   - Flash Frequency: 80MHz
   - Flash Mode: QIO
   - Flash Size: 4MB
   - Partition Scheme: Default 4MB with spiffs

Troubleshooting:
===============

Library Conflicts:
- If you have multiple versions, remove old ones
- Check Arduino/libraries folder for duplicates

Compilation Errors:
- Make sure ESP32 board is selected
- Check library versions are compatible
- Update Arduino IDE to latest version

I2C Issues:
- Check wiring connections
- Verify I2C address (DS3231 = 0x68)
- Add pull-up resistors if needed

WiFi Issues:
- Check SSID and password
- Ensure 2.4GHz network (ESP32 doesn't support 5GHz)
- Check network security settings
