## Core Libraries

### 1. RTClib by Adafruit
- **Version**: 2.1.1
- **Description**: Library for DS3231 RTC module
- **Install**: Tools > Manage Libraries > Search "RTClib"
- **URL**: https://github.com/adafruit/RTClib

### 2. ArduinoJson by Benoit Blanchon
- **Version**: 6.21.3
- **Description**: JSON library for Arduino
- **Install**: Tools > Manage Libraries > Search "ArduinoJson"
- **URL**: https://github.com/bblanchon/ArduinoJson

### 3. ESP32Servo by Kevin Harrington
- **Version**: 1.1.0
- **Description**: Servo library for ESP32
- **Install**: Tools > Manage Libraries > Search "ESP32Servo"
- **URL**: https://github.com/madhephaestus/ESP32Servo

## Built-in Libraries (No Installation Required)

### 4. WiFi
- **Description**: WiFi library for ESP32
- **Include**: `#include <WiFi.h>`

### 5. WebServer
- **Description**: Web server library for ESP32
- **Include**: `#include <WebServer.h>`

### 6. Wire
- **Description**: I2C communication library
- **Include**: `#include <Wire.h>`
