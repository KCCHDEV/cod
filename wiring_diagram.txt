Connections:
============

1. RTC DS3231 Module:
   ┌─────────────────┐
   │   DS3231 RTC    │
   │                 │
   │ VCC  ────────── ESP32 3.3V
   │ GND  ────────── ESP32 GND
   │ SDA  ────────── ESP32 GPIO21
   │ SCL  ────────── ESP32 GPIO22
   └─────────────────┘

2. 4-Channel Relay Module:
   ┌─────────────────────────────────┐
   │      4-Channel Relay Module     │
   │                                 │
   │ VCC  ────────────────────────── ESP32 3.3V (or external 5V)
   │ GND  ────────────────────────── ESP32 GND
   │ IN1  ────────────────────────── ESP32 GPIO5  (Relay 1)
   │ IN2  ────────────────────────── ESP32 GPIO18 (Relay 2)
   │ IN3  ────────────────────────── ESP32 GPIO19 (Relay 3)
   │ IN4  ────────────────────────── ESP32 GPIO21 (Relay 4)
   │                                 │
   │ NO1  ── Load 1 (Pump/Light)     │
   │ COM1 ── Power Supply            │
   │ NC1  ── (Not Connected)         │
   │                                 │
   │ NO2  ── Load 2 (Pump/Light)     │
   │ COM2 ── Power Supply            │
   │ NC2  ── (Not Connected)         │
   │                                 │
   │ NO3  ── Load 3 (Pump/Light)     │
   │ COM3 ── Power Supply            │
   │ NC3  ── (Not Connected)         │
   │                                 │
   │ NO4  ── Load 4 (Pump/Light)     │
   │ COM4 ── Power Supply            │
   │ NC4  ── (Not Connected)         │
   └─────────────────────────────────┘

3. Power Supply Connection:
   ┌─────────────────┐
   │  5V/2A Power    │
   │   Supply        │
   │                 │
   │ +5V ─────────── Relay Module VCC
   │ GND ─────────── Relay Module GND
   │                 │
   │ +5V ─────────── Relay COM1-4
   └─────────────────┘

4. Load Connections (Example for Water Pump):
   ┌─────────────────┐
   │  Water Pump     │
   │                 │
   │ + ───────────── Relay NO1
   │ - ───────────── Relay COM1
   └─────────────────┘

Complete Circuit:
================

ESP32 ── I2C ── DS3231 RTC
  │
  ├── GPIO5  ── Relay IN1 ── Load 1
  ├── GPIO18 ── Relay IN2 ── Load 2
  ├── GPIO19 ── Relay IN3 ── Load 3
  └── GPIO21 ── Relay IN4 ── Load 4

Power Supply:
5V ── Relay VCC
5V ── Relay COM1-4
GND ── ESP32 GND
GND ── Relay GND
GND ── DS3231 GND

Notes:
======
- Use 3.3V for ESP32 and RTC
- Use 5V for Relay Module (if required)
- Relay logic: LOW = ON, HIGH = OFF
- I2C address for DS3231: 0x68
- Pull-up resistors may be needed for I2C
- Use appropriate wire gauge for power connections
- Add fuses for safety in production use
